Welcome

This website holds the sole purpose of making the draconic conlang more intuitively accessible for everyone.

Draconic is a conlang made by the youtuber Human1011 to whom all the rights lay, and includes quite a few different and original elements than makes it stand out - the most notable example being the gender system - that we will be exploring in this sheet.

This is in essence just a reworked spreadsheet, but it is set up in a way which i personally would find to be more intuitive, and less workintensive in use.


This is a fan-project, and if you have anything you would like to see added to this website, if you see anything is out of date, if you see any mistakes in any way, or if you just want to say hi - then don't hesisitate to reach out to me on Discord.
