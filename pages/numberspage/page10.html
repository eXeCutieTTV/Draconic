Numbers form 0 to 16 are straight forward, they are literaly in the table ^^^

Problems start from 17, now we have to use suffixes. We do know that sē is multiple of 8, so we can add suffixes to it.
For example 17 be sē + u, e.g. (16) + (+1). So 17 = sēu, 18 = sēeχ, 19 = sēo, 20 = sēys, 21 = sēaq, 22 = sēga, 23 = sēyg.
Oh no! We ran out of suffixes! Luckely we have another multiple of 8 - fōrz. So 24 = fōrz, 25 = fōrzu, you get the point.

Now we are crossing the last multiple of 8: lān, what comes after 71 = lānyg? We can't add more, right?.. Right?
Si! That's what will help us! By using si we can add numbers to numbers! for example 72 = lān si xēχ (64 + 8), 73 = lān si χyz (64 + 9) and so on, untill 128.

While 127 is lān si hōzyg, 128 is NOT lān si lān, instead we multiply numbers, because lān is multiple of 64!
So 128 = eχ lān (2 * 64), 129 = eχ lānu (2 * 64 + 1), and so on.

Now we can start think about powers: after 511 (hog lān si hōzyg) comes 512 = lāran.
Do you know what goes next? Yes, you guessed it, 513 = lāranu, it goes on and on.
