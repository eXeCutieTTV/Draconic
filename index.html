1. Phonology

This is the second sheet.

Vowel Allophony

- all vowels have retracted tongue root before or after laryngeal consonants

- short vowels are slightly more lax and central than long vowels

- æ, a realized as [æ], [ä] in front of front consonants

- u, ū, o, and ō become [ʉ̜], [ʉ̜ː], [ɵ̜], and [ɵ̜ː] before post-alveolar consonants

- u, ū, o, and ō are often slightly unrounded

- y often elided adjacent to a sonorant when unstressed

- Pyric vowels are often realized with breathy voice, especially after voiceless fricatives, uvulars, laryngeals, and word-finally

Stress

- stress of a word is initially determined by it's root, ignoring all prefixes/suffixes

- stress is on the final syllable of the root if it is a closed syllable AND contains a long vowel

- otherwise, stress is on the penult of the root if it is a closed syllable OR contains a long vowel

- otherwise, stress is on the antepenult of the root

- if rules determine that the stress falls outside the root, stress can land on prefixes. If there are none, stress appears on the first syllable

- the only exceptions to the stress rule are for the copula lur, explain in the sheet named "To Be"

Consonants Allophony

/t̪/ → [t̪̚] if word-finally or before obstruent

/k/ → [c] if before i, ī; [ɡ̥] if intervocalically; [k̚] if word-finally or before obstruent

/q/ → [ɢ̥] if intervocalically; [q̚] if word-finally or before obstruent

/ʡ/ → [ʡ̆] if intervocalically; [ʡ̚] if word-finally or before obstruent

/ʔ/ → [ʔ̞] if intervocalically; [ʔ̚] if word-finally or before obstruent

/t͡s̠/ → [t͡ʃ] if before i, ī

/k͡x/ → [c͡ç] if before i, ī

/ɸ/ → [f] if adjacent to th; [β] if intervocallically or adjacent to z, g; [v] if adjacent to d

/s̠/ → [ʃ] if before i, ī

/z̠/ → [ʒ] if before i, ī

/x/ → [ç] if before i, ī

/ɣ/ → [ʝ] if before i, ī

/l/ → [l̪] if word-initially and before voiceless obstruents

/r/ → [r̥] if word-initially and before voiceless obstruents

/ɬ/ → [ɮ] if intervocalically

/m/ → [m̥] if word-initially and before voiceless obstruents

/n/ → [n̥] if word-initially, word-finally, and before voiceless obstruents; [n̪] if before dentals; /ŋ/ if before velars and uvulars

/ŋ/ → [ŋ̊] if before voiceless velars; [ɴ̥] if before uvulars
