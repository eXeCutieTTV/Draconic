Welcome

This website holds the sole purpose of making the draconic conlang more intuitively accessible for everyone.

Draconic is a conlang made by the youtuber Human1011 to whom all the rights lay, and includes quite a few different and original elements than makes it stand out - the most notable example being the gender system - that we will be exploring in this sheet.

This is in essence just a reworked spreadsheet, but it is set up in a way which i personally would find to be more intuitive, and less workintensive in use.


This is a fan-project, and if you have anything you would like to see added to this website, if you see anything is out of date, if you see any mistakes in any way, or if you just want to say hi - then don't hesisitate to reach out to me on Discord.

/t̪/ → [t̪̚] if word-finally or before obstruent

/k/ → [c] if before i, ī; [ɡ̥] if intervocalically; [k̚] if word-finally or before obstruent

/q/ → [ɢ̥] if intervocalically; [q̚] if word-finally or before obstruent

/ʡ/ → [ʡ̆] if intervocalically; [ʡ̚] if word-finally or before obstruent

/ʔ/ → [ʔ̞] if intervocalically; [ʔ̚] if word-finally or before obstruent

/t͡s̠/ → [t͡ʃ] if before i, ī

/k͡x/ → [c͡ç] if before i, ī

/ɸ/ → [f] if adjacent to th; [β] if intervocallically or adjacent to z, g; [v] if adjacent to d

/s̠/ → [ʃ] if before i, ī

/z̠/ → [ʒ] if before i, ī

/x/ → [ç] if before i, ī

/ɣ/ → [ʝ] if before i, ī

/l/ → [l̪] if word-initially and before voiceless obstruents

/r/ → [r̥] if word-initially and before voiceless obstruents

/ɬ/ → [ɮ] if intervocalically

/m/ → [m̥] if word-initially and before voiceless obstruents

/n/ → [n̥] if word-initially, word-finally, and before voiceless obstruents; [n̪] if before dentals; /ŋ/ if before velars and uvulars

/ŋ/ → [ŋ̊] if before voiceless velars; [ɴ̥] if before uvulars
